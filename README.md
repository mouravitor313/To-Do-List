# To-Do-List
Estou utilizando este projeto para por em prática o conhecimento adquirido em PHP, além de aprender coisas novas. 
Este projeto se trata de uma To-Do List feita em PHP. A proposta é criar um site onde o usuário irá se registrar, criar suas tarefas e poder editá-las ou removê-las. 
Funcionalidades como registro e criação de tarefas já foram concluídas, faltando apenas criar um login e dar a opção de editar e remover as tarefas.
